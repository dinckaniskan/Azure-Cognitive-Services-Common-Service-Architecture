# Debug Functions Locally

# Sample App Locally

`
cd SampleApp
`

### Create Virtual Env and Activate:

`
python -m venv .venv
source .venv/Scripts/activate 
`

### Install dependencies:

`
pip install -r requirements.txt
`

### Run App:

`
: streamlit run app.py
`
